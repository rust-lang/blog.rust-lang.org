The 1.97.0 pre-release is ready for testing. The release is scheduled for
July 9. [Release notes can be found here.][relnotes]

You can try it out locally by running:

```
RUSTUP_DIST_SERVER=https://dev-static.rust-lang.org rustup update stable
```

The index is <https://dev-static.rust-lang.org/dist/2026-07-07/index.html>.

You can leave feedback on the [internals thread](https://internals.rust-lang.org/t/rust-1-97-0-pre-release-testing/24439).

The release team is also thinking about changes to our pre-release process:
we'd love your feedback [on this GitHub issue][feedback].
