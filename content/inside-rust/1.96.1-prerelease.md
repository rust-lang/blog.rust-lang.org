The 1.96.1 pre-release is ready for testing. The release is scheduled for
June 30. [Release notes can be found here.][relnotes]

You can try it out locally by running:

```
RUSTUP_DIST_SERVER=https://dev-static.rust-lang.org rustup update stable
```

The index is <https://dev-static.rust-lang.org/dist/2026-06-27/index.html>.

You can leave feedback on the [internals thread](https://internals.rust-lang.org/t/rust-1-96-1-pre-release-testing/24417).

The release team is also thinking about changes to our pre-release process:
we'd love your feedback [on this GitHub issue][feedback].
